# 学习 Table API

{{< hint warning >}}
**注意:** 此页面需要翻译。请参阅英文版本获取完整内容：[Learn the Table API]({{< ref "docs/learn-flink/table_api" >}})
{{< /hint >}}

本培训的重点是广泛介绍 Table API，使您能够开始编写流分析应用程序。

Table API 是 Flink 的声明式关系 API。您描述*想要*计算什么，Flink 会找出*如何*高效地计算它。
相同的查询可以在批处理和流数据上运行而无需修改，Flink 的优化器会自动选择高效的执行计划。
