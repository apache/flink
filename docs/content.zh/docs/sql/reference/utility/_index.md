# Utility Statements

Utility statements provide information about database objects and help manage the SQL environment.

- **[SHOW Statements]({{< ref "docs/sql/reference/utility/show" >}})**: List database objects
- **[DESCRIBE]({{< ref "docs/sql/reference/utility/describe" >}})**: Display metadata about tables and views
- **[EXPLAIN]({{< ref "docs/sql/reference/utility/explain" >}})**: Show the execution plan for a query
- **[USE]({{< ref "docs/sql/reference/utility/use" >}})**: Set the current catalog or database
- **[SET & RESET]({{< ref "docs/sql/reference/utility/set" >}})**: Configure session properties
- **[LOAD & UNLOAD]({{< ref "docs/sql/reference/utility/load" >}})**: Manage modules
- **[ANALYZE]({{< ref "docs/sql/reference/utility/analyze" >}})**: Collect table statistics
- **[JAR Statements]({{< ref "docs/sql/reference/utility/jar" >}})**: Manage JAR files
- **[JOB Statements]({{< ref "docs/sql/reference/utility/job" >}})**: Manage Flink jobs
- **[CALL]({{< ref "docs/sql/reference/utility/call" >}})**: Execute stored procedures
