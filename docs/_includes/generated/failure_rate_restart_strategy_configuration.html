Key Default Description
restart-strategy.failure-rate.delay
 "0 s" Delay between two consecutive restart attempts when using `FailureRateRestartStrategy`. It can be specified using Scala's `FiniteDuration` notation: "1 min", "20 s"
restart-strategy.failure-rate.failure-rate-interval
 "1 min" Time interval for measuring failure rate when using `FailureRateRestartStrategy`. It can be specified using Scala's `FiniteDuration` notation: "1 min", "20 s"
restart-strategy.failure-rate.max-failures-per-interval
 1 Maximum number of restarts in given time interval before failing a job when using `FailureRateRestartStrategy`.
