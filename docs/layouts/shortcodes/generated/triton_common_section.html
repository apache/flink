Option Required Default Type Description
provider
 required (none) String Specifies the model function provider to use, must be 'triton'.
endpoint
 required (none) String Full URL of the Triton Inference Server endpoint, e.g. http://localhost:8000/v2/models.
model-name
 required (none) String Name of the model to invoke on Triton server.
model-version
 required latest String Version of the model to use. Defaults to 'latest'.
timeout
 optional 30000 Long Request timeout in milliseconds.
